# Portal Gestión RF DGM

Esta es una primera maqueta editable de un portal web simple para gestión de Riesgos Fatales, RESSO, auditorías, capacitaciones y documentos.

## Cómo abrirlo en tu computador

1. Descarga el archivo ZIP.
2. Descomprime la carpeta.
3. Abre el archivo `index.html` con doble clic.
4. Se abrirá en Chrome, Edge o el navegador que uses.

## Qué archivos contiene

- `index.html`: estructura de la página.
- `estilos.css`: diseño visual, colores, tamaños y distribución.
- `script.js`: datos de ejemplo, buscador y tarjetas dinámicas.
- `README.txt`: esta guía.

## Cómo editar textos simples

Puedes abrir `index.html` con Bloc de notas, Visual Studio Code o cualquier editor de texto.

Ejemplos:
- Cambiar título: buscar `Gestión RF DGM`.
- Cambiar módulos: buscar las secciones `RESSO V10`, `Capacitaciones`, `Auditorías`.
- Cambiar datos de ejemplo: editar el archivo `script.js`.

## Cómo publicarlo en GitHub Pages

1. Crear un repositorio en GitHub.
2. Subir estos archivos a la raíz del repositorio.
3. Ir a Settings > Pages.
4. En Source, elegir "Deploy from a branch".
5. Seleccionar rama `main` y carpeta `/root`.
6. Guardar.

Luego GitHub entregará una URL pública del sitio.
